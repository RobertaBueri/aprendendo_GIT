Aprendendo a usar GIT

    Alguns comandos importantes:

      Instalação e inicialização
    • git -V: usado para baixar o git
    • git --version: usado para ver a instalação
    • git init: inicializa o git no projeto
      ver as pendências
    • git status: mostra as pendências com o git
      Salva os arquivos:
    • git add. :adiciona todos os arquivos no git
    • git commit -a -m "nome da modificação": salva o arquivo a partir do momento atual
      integração com git hub
    • git pull : manda o arquivo do remoto para o local
    • git checkout -b"nome": cria um branch novo
    • git checkout nome: se move para o branch novo
    • git branch: mostra a lista das branchs e em qual você está
    • git merge nome: comando para unir as alterações das branchs
    • git log: mostra todos os commics feitos para ter uma visão geral do projeto
